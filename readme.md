## **Scrollable map**

**Setup**

    npm install

**run dev server**

    npm run dev

**build**

    npm run build

---

**Example HTML**

    <div id="properties_container" class="articles">
    	<article id="property-1" data-coo="-12.1008799,-77.0280677" 	data-text="Home" >
    		<h1>Home</h1>
    	</article>
    </div>
    <div id="interactive_map" data-init-coo="-12.100867,-77.034787" data-init-zoom="17"></div>

- For address->coordinates conversions, see commented out sections
